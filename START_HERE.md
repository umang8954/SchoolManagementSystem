## Project Overview

- 18 Java classes
- 15 REST endpoints
- 3 entities with CRUD operations
- Swagger/OpenAPI 3.0 documentation

- H2/MySQL database

## 🔄 How It Works: Request Flow

```
1. User creates student via API
   ↓
2. Controller receives request
   ↓
3. Service validates data
   ↓
4. Repository saves to database
   ↓
5. API returns response with created resource
```

**Result:** Clean, synchronous REST API with immediate response!
